# Usage

## vim
Add `:source <repo>/vimrc` to ~/.vim/vimrc

Uses Vim-plug

## tmux
Add the follow to ~/.tmux.conf:
```
POWERLINECONF="<wheretfeveritis>"
source "<repo>/tmux.conf"
``` 
Uses powerline

## zsh
Add the following to ~/.zshrc:
```
export ZSH="<wheretfeveritis>"
source <repo>/zshrc
```
Uses Powerlevel10k and oh-my-zsh
